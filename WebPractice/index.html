Early Days In Seconday School
My interest for software development began when I was in year 7 when I was shown Scratch in I.T class. This was an appliction that allowed users to create 2D games or animations and was my first introduction into programming. Instead of writing lines of code it used blocks; which were basically variables and commands that could be dragged-and-dropped together to create functions or methods. This made it easy for first time programmers how programming worked, but in a very simplfied way.
